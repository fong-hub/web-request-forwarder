# 更新日志

本项目所有重要变更均记录于此文件。

格式基于 [Keep a Changelog](https://keepachangelog.com/)，版本号遵循轻量级语义化版本规范。

## [0.0.4] - 2026-05-08

### 新增

- 自定义 `CustomSelect` 下拉组件，替代原生 `<select>`，解决下拉列表遮挡当前选项的问题
- 表格粘性表头（滚动时始终置顶）和斑马纹隔行背景，提升长列表可读性
- 诊断卡片增加左边框颜色标识，区分 warning 与 info 级别
- 按钮 `:active` 按压反馈与 `:focus-visible` 聚焦环，提升键盘可访问性

### 变更

- 选项页、弹出面板、错误提示、诊断信息、扩展悬浮提示等全面中文化
- 新增规则默认资源类型从 `xmlhttprequest` 调整为 `script`
- 规则编辑弹窗表头改用 `position: sticky` 固定，表单内容区独立滚动
- 匹配类型与重定向类型字段改为并排双列布局
- 补充说明字体从 `0.88rem` 缩小至 `0.8rem`，视觉层次更克制
- 空状态文本居中，颜色更柔和
- 表格操作按钮略微缩小，在窄列中更协调

### 发布产物

- `releases/request-forwarder-v0.0.4.zip`
- `releases/v0.0.4.md`
