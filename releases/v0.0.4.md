# Request Forwarder v0.0.4

Release date: 2026-05-08

## Summary

全面中文化界面，优化表单交互与表格体验，调整默认规则配置。

## Highlights

- **界面全面中文化**：选项页、弹出面板、错误提示、诊断信息、后台悬浮提示等所有用户可见文案改为中文
- **默认资源类型调整**：新增规则时默认资源类型从 `xmlhttprequest` 改为 `script`
- **编辑弹窗布局优化**：表头 `position: sticky` 固定，表单内容区滚动；匹配类型与重定向类型并排双列，减少垂直空间占用
- **自定义下拉选择器**：替换原生 `<select>` 为 `CustomSelect` 组件，下拉列表在触发器下方展开，不再遮挡当前已选项
- **表格体验提升**：增加 sticky 表头（滚动时始终可见）、斑马纹隔行背景、选项页与弹出面板分别分配合理列宽
- **视觉与交互细节**：诊断卡片增加左边框色区分 warning/info 级别；按钮增加 `:active` 按压反馈；全局增加 `:focus-visible` 聚焦环；空状态文本居中；补充说明字体缩小更克制
