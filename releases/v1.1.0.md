# Request Forwarder v1.1.0

Release date: 2026-03-27

## Summary

This release upgrades the popup workflow, adds live rule-hit feedback, and adds a scripted release preparation flow for GitHub publishing.

## Highlights

- Reworked the popup into a compact operations view with a global switch, fixed table header, content-only scrolling, and a direct entry to the full options page.
- Added rule match tracking based on DNR feedback so matched rules are highlighted in popup and options, with per-rule hit counts.
- Added action badge feedback to show matched rule totals and switched the extension icon to grayscale when the global engine is disabled.
- Improved the options page rule table and modal editor flow to support faster editing and clearer runtime status.
- Added `npm run release:prepare -- <version>` to sync versions, run validation, build artifacts, create release notes, and prepare a GitHub release command.

## Validation

- `npm run lint`
- `npm run test`
- `npm run build`

## GitHub Release Assets

- `releases/request-forwarder-v1.1.0.zip`
- `releases/v1.1.0.md`

## Publish Command

```bash
gh release create v1.1.0 releases/request-forwarder-v1.1.0.zip --title "v1.1.0" --notes-file releases/v1.1.0.md
```
