# Request Forwarder v0.0.3

发布日期：2026-03-30

## 版本概述

本次版本主要补充了对外发布所需的工程化能力，并继续优化弹窗与规则编辑体验，适合作为 `0.0.3` 正式发布版本。

## 更新内容

- 新增 GitHub Actions 自动发布工作流，支持通过 `v*` tag 自动构建并创建 GitHub Release
- 新增 GitHub issue 模板与 release 分类配置，方便后续协作和版本维护
- 新增 `CHANGELOG.md`，用于持续维护版本演进记录
- 优化编辑弹窗交互，点击遮罩不再关闭弹窗，只能通过右上角关闭按钮退出
- 移除规则编辑器中带敏感域名的信息占位文案，替换为通用示例
- 调整命中规则统计逻辑，禁用规则不再参与扩展 badge 的命中数统计
- 保持 popup 和详情页中的规则命中高亮、命中计数和状态反馈能力

## 验证结果

- `npm run lint`
- `npm run test`
- `npm run build`

## 发布产物

- `releases/request-forwarder-v0.0.3.zip`
- `releases/v0.0.3.md`

```bash
gh release create v0.0.3 releases/request-forwarder-v0.0.3.zip --title "v0.0.3" --notes-file releases/v0.0.3.md
```
