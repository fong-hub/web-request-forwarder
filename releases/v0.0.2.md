# Request Forwarder v0.0.2

发布日期：2026-03-27

## 版本概述

本次版本主要优化了扩展弹窗交互，补充了规则命中反馈能力，并新增了可复用的发布准备脚本，用于后续 GitHub Release 发布。

## 项目说明

Request Forwarder 是一个轻量级、本地部署的 Chrome 请求转发扩展，适用于本地调试、接口切换、Mock 替换和测试环境验证。它不依赖独立代理服务，规则直接在浏览器扩展中生效，适合快速部署和日常开发使用。

## 产品优势

- 轻量：无需额外代理服务或后台进程
- 本地部署：规则存储和执行都在浏览器本地完成
- 易于使用：安装扩展后即可直接配置和启用
- 低侵入：无需修改业务代码或后端服务
- 调试效率高：适合快速切换接口目标和测试环境

## 更新内容

- 重构 popup，提供更紧凑的规则查看与全局开关交互
- 新增规则命中跟踪，高亮已命中的规则并显示命中次数
- 新增扩展图标 badge 反馈，并在全局禁用时将图标置灰
- 优化详情页规则表格与弹窗编辑交互
- 新增 `npm run release:prepare -- <version>` 发布准备脚本，用于同步版本、执行校验、构建产物并生成发布说明
